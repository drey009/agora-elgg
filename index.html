Conheça o ELGG

O que é?

Então o app ELGG é uma software de código aberto, ele oferece uma espaço de blog, comunidades com fóruns de discussões ou blogs comunitários, espaço para respositórios de arquivos,e-portifólio, tecnologia RSS para conteúdo gerado dentro da rede.


O que é Plugin?

Então plugins são componemtes que podem ser desenvolvidos para extender funcionalidades do ELGG, adicionar novos recursos, alterar visões de estilo do sistema.

Um plugin é composto de um arquivo que ficam dentro de uma pasta que possue o nome plugin. Os plugins ficam dentro da pasta mod do ELGG.


VANTAGENS: Código aberto, comunidades de programadores sempre crescentes e milhares de plugins já desenvolvidos que atendem, as necessidades da maioria dos projetos dos usuários.


DESVANTAGENS: Em um desenvolvimento em HTML não contamos muito com a acessibilidade aos recursos dos aparelhos móveis.
